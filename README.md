Project description

Install
-------
```commandline
pip install ocli
```

Development
-----------
We need installed `pyenv` and `pipenv`.
```console
git clone git@github.com:voronind/ocli.git

cd ocli
pipenv install --dev
```

Run tests:
```console
pipenv run fulltest
```
